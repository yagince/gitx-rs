# gitx-rs

my git subcommands by Rust

# Requirements

- Rust 1.9.0

## Installation

You need to add the `.cargo/bin` to your PATH

```
cargo install --git https://github.com/yagince/gitx-rs.git
```

## Commands

### git start

create a new branch with branch-type and issue-number
